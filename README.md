# Stroke Mitra Prototype

Early Stroke Detection Screening Prototype (PWA).

## Features
- **Face Analysis**: Detects facial alignment and symmetry.
- **Voice Check**: analyzes speech clarity.
- **Motion Test**: Tests coordination using device sensors.

## Tech Stack
- Frontend: React + Vite + Vanilla CSS
- Backend: Node.js + Express (Minimal)
- PWA: Vite PWA Plugin

## How to Run

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server (runs both Client and API):
   ```bash
   npm run start
   ```

3. Open your browser at `http://localhost:5173`.
   - On mobile, connect to the same Wi-Fi and use your computer's IP address (e.g., `192.168.1.X:5173`) to test sensors properly.

## Note
This is a PROTOTYPE for screening only. Not a medical device.
